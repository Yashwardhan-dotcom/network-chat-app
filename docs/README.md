##  Project Overview

A multi-client chat application using socket programming to demonstrate networking concepts like TCP/IP and client-server architecture.

---

##  How It Works

1. The server listens for incoming client connections using a specific IP address and port.
2. Multiple clients connect to the server using sockets.
3. When a client sends a message, it is transmitted to the server.
4. The server broadcasts the message to all connected clients.

---

##  Data Flow

Client → Server → All Clients

---

##  Technologies Used

* Python
